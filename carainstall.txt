download node.js
1. klik link https://nodejs.org
2. klik "dapatkan node.js"
3. Pilih yang LTS saja karena lebih stabil untuk belajar dan proyek UKK Klik Download Node.js (LTS).
4. Setelah download selesai:

-Buka file .msi yang tadi didownload
-Klik Next
-Centang I accept the terms
-Klik Next terus saja
-Klik Install
-Tunggu sampai selesai
-Klik Finish
5.buka cmd lalu cek apakah udah terinstall dengan ketik 'node -v' dan 'npm -v' karena biasa nya npm udah termasuk e node.js

download git
1. klik link https://git-scm.com
2. klik download for windows
3.Setelah download selesai:

-Buka file Git-xx.x.x-64-bit.exe
-Klik Next
-Pada semua pilihan biarkan default saja
-Klik Next terus
-Klik Install
-Tunggu sampai selesai
-Klik Finish
4. lalu cek di cmd dengan ketik 'git --version'

download composer
1. klik link https://getcomposer.org
2. jalankan Composer-Setup.exe lalu next
3. akan muncul pilihan php path, (karena pakai xampp) C:\xampp\php\php.exe
4.Kalau tidak muncul otomatis:

klik Browse
cari folder xampp
masuk ke php
pilih php.exe
5.next, install, dan finish
6. cek di cmd dengan ketik 'composer -v'
